ERA Mail Serverخادم بريد متكامل سيادي

خادم بريد وتعاون محلي — ليس Postfix + Dovecot. مكدس Rust + Go متكامل (معمارية من فئة Stalwart، تنفيذ ERA). بديل لـ Exchange وCommuniGate وIceWarp بدون موصل Outlook على العميل — توافق Exchange من الخادم عبر Autodiscover وEWS.

توافق Outlook والجوال (من الخادم)

  • Autodiscover + EXCH — Outlook يرى ملف Exchange؛ بدون إضافة عميل.
  • EWS subset — بريد، تقويم، جهات اتصال، ملاحظات/مهام (مجموعة فرعية موثقة).
  • CalDAV — تقويم كامل للقراءة/الكتابة.
  • CardDAV — جهات اتصال مؤسسية.
  • ActiveSync subset — مزامنة iOS/Android مثل Exchange.
  • SMTP/IMAP + TLS — عملاء قياسيون.

قدرات رئيسية

  • نواة بريد متكاملة — منتج واحد، air-gap.
  • تدقيق ClickHouse — أثر SIEM إلزامي.
  • تراخيص Ed25519 دون اتصال؛ هدف 60,000+ صندوق بريد.

المعمارية التقنية

  • Rust — era-mail-core: SMTP/IMAP.
  • Go — mail-api: Autodiscover، EWS، CalDAV، CardDAV، ActiveSync.
  • أنماط Stalwart — كود ERA أصلي، بدون fork.
