ERA Mail ServerSovereign integrated mail backend

Native on-prem mail and collaboration backend. Integrated Rust + Go stack — single product, air-gap ready. Client compatibility is implemented on the server via Autodiscover, EWS, CalDAV, CardDAV, and ActiveSync — no desktop plugins or client-side connectors.

Outlook & mobile parity (server-side)

  • Autodiscover + EXCH — Outlook uses standard EXCH Autodiscover profile; no client plugin.
  • EWS subset — mail, calendar, contacts, notes/tasks (documented subset).
  • CalDAV — full calendar read/write (Apple Calendar, Thunderbird).
  • CardDAV — corporate contacts (gov pilot scope).
  • ActiveSync subset — iOS/Android mobile sync (gov pilot scope).
  • SMTP/IMAP + TLS — standard clients and mass deployment.
