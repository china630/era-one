ERA Mail ServerSovereign integrated mail backend

Native on-prem mail and collaboration backend — not Postfix + Dovecot. Integrated Rust + Go stack (Stalwart-class architecture, ERA implementation). Replaces Exchange, CommuniGate and IceWarp without an Outlook desktop connector — server-side Exchange compatibility via Autodiscover and EWS.

Outlook & mobile parity (server-side)

  • Autodiscover + EXCH — Outlook sees an Exchange profile; no client plugin.
  • EWS subset — mail, calendar, contacts, notes/tasks (documented subset; not full MAPI).
  • CalDAV — full calendar read/write (Apple Calendar, Thunderbird).
  • CardDAV — corporate contacts (gov pilot scope).
  • ActiveSync subset — iOS/Android sync like Exchange (gov pilot scope).
  • SMTP/IMAP + TLS — standard clients and mass deployment.

Key capabilities

  • Integrated mail core — single product, air-gap ready.
  • ClickHouse audit — mandatory SIEM trail (ERA Control integration).
  • Offline Ed25519 licensing; scale target 60,000+ mailboxes.
  • Zero-Touch Autodiscover for Outlook and mobile.

Technical architecture

  • Rust — era-mail-core: SMTP/IMAP delivery.
  • Go — mail-api: Autodiscover, EWS, CalDAV, CardDAV, ActiveSync adapters.
  • Patterns from Stalwart (integrated protocols) — native ERA code, no fork.

Co-editing attachments requires ERA Office (separate license). Hybrid transition: ERA Mail Connect.
