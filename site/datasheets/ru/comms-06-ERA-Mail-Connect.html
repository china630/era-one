ERA Mail ConnectMigration tier

Webmail и desktop/mobile клиент с BFF к существующему IMAP/JMAP/EWS серверу. Быстрая замена клиента без миграции backend. Не заменяет ERA Mail Server в Full Suite GA narrative.

Когда использовать

  • Пилот webmail при сохранении Exchange / CommuniGate / IceWarp.
  • Поэтапная миграция: Connect → Mail Server.
  • Ограничения: нет полного набора Server (calendar/ActiveSync на уровне native backend).
