ERA PAMPrivileged access

Enterprise vault for administrator passwords and secrets, access checkout per policy, secure SSH/RDP proxy, and privileged session recording. A separate edition for perimeters where admin access control is critical (analogous to Password Manager Pro).

Purpose

Eliminate "shared admin passwords" and unaccounted server access. All privileged user actions are transparent for audit.

Key capabilities

  • Enterprise password and key vault.
  • Credential checkout per policy.
  • SSH/RDP via proxy with session recording.
  • HSM integration readiness for demanding perimeters.

Who it's for

Banks, public sector, critical infrastructure operators with privileged access policies.

Licensing

Separate ERA PAM license — standalone edition, not included in the base XDR bundle.
